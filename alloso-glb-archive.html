배포 가이드 · GitHub + Netlify (무료 · 영구)
01
GLB 드롭 → 치수 확인 → GLB URL 입력 → manifest.json 저장
02
GitHub 저장소 루트에 manifest.json 커밋
GLB 파일은 /assets/ 폴더 또는 Cloudflare R2에 업로드
03
Netlify → GitHub 저장소 연결 → 자동 배포 완료
이후 git push 할 때마다 자동 반영 (~30초)
04
새 소파 추가: GLB 드롭 → manifest.json 저장 → GitHub push → 끝
GLB 파일 저장 위치
옵션 A · GitHub /assets/ 폴더 — 파일당 100MB 이하, 무료. URL: /assets/sofa-name.glb
옵션 B · Cloudflare R2 — 무료 10GB/월, 대용량 권장. URL: https://pub-xxx.r2.dev/sofa.glb
manifest.json의 "glb" 필드에 URL을 입력하면 Living Studio가 자동 연결합니다.
